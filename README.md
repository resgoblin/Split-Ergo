# Split-Ergo
This is a collection of cases I have modeled to house the PCBs distributed by Typeractive.xyz

All versions will be modeled to fit a 700 MAH LiPo beneath the PCB, this is just about the largest cell that can fit my layout of choice (Choc Corne 6 Col) without modifying the battery leads. If you need/want a larger battery I can be found in the typeractive discord server and may be able/willing to accomodate.

There are 2 versions of tenting that will be utilized, a 3 tier/setting foot and a magsafe compatible metal ring.
